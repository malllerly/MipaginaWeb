APRENDEREMOS HTML Y CSS

CLASE 1

Introducción de Visual estudio code, etiquetas principales, elementos con los que conforma un HTML y el concepto de anidamiento.

CLASE 2

Herramientas a utilizar para crear una página web (HYML, CSS y JavaScript) y algunas etiquetas de HTML como UL, OL, LI, A y IMG.

CLASE 3

Aprendimos etiquetas de HTML como Ploren , B, i, SUB, SUP, BR/, HR/, STRONG, EM, BLOCKQUETE, ABBR, CITE y DFN.

CLASE 4

Aprendimos a utilizar las etiquetas IMG SRC, DIV, SPAN Y VIDEO SRC en HTML. Introducción a CSS, con elementos fundamentales para diseñar la pagina como COLOR, BACKGROUND COLOR, ID, FONT y enlace de páginas.

CLASE 5

Repasamos lo aprendido en CSS y lo pusimos en práctica.

CLASE 6

CLASE 7

    CLASE 8
